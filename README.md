codezoo
=======

A collaborative online IDE for beginner programmers


### Building 

```
git pull https://github.com/meadowstream/codezoo.git
cd codezoo
sudo npm install
```

### Running

`node server.js [port number | default: 8080]`

Then, navigate to `localhost:8080/`
